Explicação do Trabalho

Neste trabalho foi criado uma aplicação em JavaScript que faz operações matemáticas com matrizes.

As matrizes são geradas aleatoriamente toda vez que o usuário atualiza a página (F5) e é feito o cálculo de acordo com o que eu escolhi, que seria a adição e a multiplicação das matrizes.

Em JavaScript, não existe uma coisa chamada "array multidimensional" pronta para usar. Então, a gente cria um array que dentro dele tem outros arrays. Isso vira o nosso array multidimensional.

Foi usado Math.random() para gerar os números aleatórios de 0 até 100, junto foi utilizado o Math.ceil() para arredondar o número para cima.

Foi utilizado HTML para estruturar a página.Também foi utilizado CSS para estilizar a página, deixando-a mais bonita e organizada, onde as tabelas que contém as matrizes são todas modificadas, o usuário passa o mouse sobre as células da tabela e a borda e o fundo muda de cor e etc. E é claro, o JavaScript foi usado para calcular as matrizes.

Sobre os cálculos:

Adição de Matrizes: Na adição pegamos cada elemento da primeira matriz e somamos com o elemento correspondente da segunda matriz, colocando em uma nova matriz para guardar os resultados e exibi-los depois.

Multiplicação de Matrizes: Já na multiplicação nós pegamos cada linha da primeira matriz e multiplicamos pelos elementos de cada coluna da segunda matriz, somando os resultados, depois fazemos como na adição, armazenamos em uma outra matriz para exibir os resultados depois.
